# 跨浏览器打印方案

#### 项目介绍
跨浏览器打印的控件网上很多，露肚皮不支持PDF打印，jatools授权费太高。。。。
所以那么自己动手吧，反正也是内部使用，可以要求每台电脑都要安装jre，实在不想把jre打到可执行包里去，那样略大。
经过测试，这个套路是行得通的，现在放出来给用得着的朋友分享下。
